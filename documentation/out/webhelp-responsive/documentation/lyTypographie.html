Typographische Besonderheiten

Für alle Element-Bereiche (<Word>, <TextLine> und <TextRegion>) gelten die folgenden typographischen Auszeichnungswerte.

Vorrangig für die Dokumentation von typographischen Merkmalen ist das Element <TextStyle> zu verwenden.

Parallel kann das custom Attribut innerhalb von <Word>, <TextLine> und <TextRegion> verwendet werden. In einigen Fällen kann nur mit dem custom-Attribut eine typographische Besonderheit dokumentiert werden. Es sind folgende @custom-Attribut-Werte verfügbar.
